### **Final Team Project: MERN Stack Web Application**

**Total Points: 250\**
**Team Size: 4 Students**

### **Overview:**

Your team will build a full-stack web application using the **MERN
stack** (MongoDB, Express, React, Node.js). The application must
demonstrate proper full-stack integration and provide full CRUD
functionality via a web interface.

This project is designed to assess your ability to build a functional
web app from the ground up. All members must contribute to the
front-end, back-end, and database portions. Peer reviews will determine
individual participation and impact final grades.

**Deployment is not required**, but up to **10 bonus points** will be
awarded for teams who successfully deploy their application.

### **Project Requirements & Point Breakdown (250 points):**

#### **1. Frontend -- React (60 points)**

- Uses **React Router** for routing across multiple pages/components --
  **15 pts\**

- Uses **Axios** to send and retrieve data from the backend -- **15
  pts\**

- Interface is clean, responsive, and user-friendly (CSS or Bootstrap
  acceptable) -- **15 pts\**

- Uses state management properly via useState, useEffect, and hooks --
  **15 pts\**

#### **2. Backend -- Node.js & Express (50 points)**

- Express server is structured and runs without errors -- **10 pts\**

- Routes properly handle all expected requests and responses -- **15
  pts\**

- Includes error handling and basic input validation -- **10 pts\**

- Proper connection and interaction with MongoDB -- **15 pts\**

#### **3. Database -- MongoDB with Mongoose (40 points)**

- At least **2 well-defined Mongoose models** with proper schema and
  validation -- **10 pts\**

- Full implementation of CRUD (Create, Read, Update, Delete) -- **20
  pts\**

- Initial data or seed functionality is available for testing -- **10
  pts\**

#### **4. Integration and Functionality (30 points)**

- End-to-end flow is functional: React ↔ Axios ↔ Express ↔ MongoDB --
  **15 pts\**

- Application solves a real-world use case with **3 or more distinct
  features** -- **15 pts\**

#### **5. Documentation and Video Presentation (20 points)**

- **YouTube video** (5--10 minutes) uploaded to a team member's account
  and shared via link:

  - Walkthrough of the app

  - Overview of each part of the stack

  - Explanation of how all requirements were met

  - Every student must speak in the presentation -- **10 pts\**

- **PowerPoint or slide deck** must be included and referenced in the
  video -- **10 pts\**

#### **6. Peer Review (50 points -- individual grade)**

Each student will submit a **confidential peer review** for their
teammates covering:

- Contributions to the project

- Communication and reliability

- Technical effort and teamwork

**No peer review submission = automatic 0/50** on this section.

### **Bonus (Optional -- up to 10 points)**

- Functional deployment using **Render**, **Vercel**, or similar -- **up
  to 10 bonus pts\**
